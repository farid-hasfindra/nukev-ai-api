# Nukev AI Backend API
AI-powered Spoiler Shield for Nukev short video platform.

## Setup
Backend ini dideploy menggunakan Docker SDK di Hugging Face Spaces.

### Environment Variables
- `GROQ_API_KEY`: API Key untuk Groq Cloud (Llama-3.1-8B).
- `BACKEND_CORS_ORIGINS`: Daftar URL yang diizinkan untuk mengakses API (misal: Vercel app).
